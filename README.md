# OrteZA Smart — Firmware V20.0

Cod Arduino pentru orteza inteligentă de monitorizare posturală.

## Hardware necesar
- ESP32-C6 Waveshare
- 3x BNO086 SparkFun
- Multiplexor TCA9548A
- Bandă LED WS2812B 60 LED-uri
- Buzzer activ

## Biblioteci necesare
- SparkFun BNO080 Arduino Library
- Adafruit NeoPixel
- ESP32 BLE Arduino

## Utilizare
1. Instalează bibliotecile în Arduino IDE
2. Selectează board: ESP32C6 Dev Module
3. Upload și conectează prin Bluetooth
